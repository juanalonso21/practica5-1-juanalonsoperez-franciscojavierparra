# VueDining - Reservas de Restaurante

Documentación del Ejercicio 3. Un sistema avanzado de selección de mesas y turnos horarios para un restaurante.

## Funcionalidades Clave

- **Selector de Turnos**: Selección de fecha y hora.
- **Plano del Restaurante**: Mapa interactivo de las mesas.
- **Estado de Mesas**: Diferenciación visual entre mesas libres, seleccionadas y ocupadas.
- **Reservas según Capacidad**: Filtrado de disponibilidad basado en el número de comensales.
