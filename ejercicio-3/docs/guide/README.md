# Guía de Reservas VueDining

Reservar una mesa en nuestro restaurante es un proceso visual y sencillo.

## Proceso de Reserva

1. **Selección de Turno**: Elige la fecha y el tramo horario disponible.
2. **Plano de Mesas**: Haz clic en una mesa libre (verde) para seleccionarla. Las mesas ocupadas (rojo) no pueden seleccionarse.
3. **Confirmación**: Una vez seleccionada la mesa y el turno, confirma tu reserva.

## Leyenda de Colores

- **Verde**: Mesa disponible para reserva.
- **Rojo**: Mesa ya ocupada por otro cliente.
- **Azul**: Tu mesa seleccionada actualmente.
