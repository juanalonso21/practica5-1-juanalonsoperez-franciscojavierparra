# Checkout de Tienda Online

Esta es la documentación del Ejercicio 2. Se trata de un flujo de checkout multipaso implementado con Vue 3, Vee-Validate y Pinia.

## Tecnologías Utilizadas

- **Vue 3**: Framework principal.
- **Pinia**: Gestión del estado del carrito y datos de usuario.
- **Vee-Validate & Yup**: Validación robusta de formularios.
- **Vue Router**: Navegación entre los pasos del checkout.
