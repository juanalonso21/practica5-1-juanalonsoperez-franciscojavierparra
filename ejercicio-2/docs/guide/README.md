# Guía del Asistente de Checkout

El proceso de compra está dividido en varios pasos lógicos para mejorar la experiencia del usuario.

## Pasos del Checkout

1. **Carrito**: Revisión de los productos seleccionados y cantidades.
2. **Datos de Envío**: Formulario para la dirección de entrega y contacto.
3. **Método de Pago**: Selección de la forma de pago (Tarjeta, PayPal, etc.).
4. **Confirmación**: Resumen final antes de realizar el pedido.

## Validación

Cada campo cuenta con validación en tiempo real. No se permite avanzar al siguiente paso si existen errores en el formulario actual.
