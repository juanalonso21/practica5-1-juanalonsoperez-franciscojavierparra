# Reserva de Eventos

Bienvenido a la documentación del Ejercicio 1. Esta aplicación permite gestionar reservas de eventos de forma intuitiva, con validación de datos y soporte para temas claro y oscuro.

## Características

- Formulario reactivo con validación.
- Soporte para modo oscuro.
- Barra de progreso de completado.
- Resumen dinámico de la reserva.
