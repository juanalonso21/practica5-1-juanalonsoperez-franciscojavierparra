# Guía de Usuario

Esta guía te ayudará a entender cómo funciona el sistema de reserva de eventos.

## Uso del Formulario

1. **Datos Personales**: Rellena tu nombre, NIF, teléfono y correo electrónico. El sistema validará el formato automáticamente.
2. **Detalles del Evento**: Selecciona el tipo de evento, la fecha, la hora y el número de asistentes.
3. **Servicios Adicionales**: Elige las opciones de catering que prefieras.
4. **Resumen**: Observa cómo el resumen se actualiza en tiempo real mientras completas el formulario.

## Temas

Puedes alternar entre el modo claro y oscuro utilizando el botón ☀️/🌙 en la parte superior derecha de la aplicación.
