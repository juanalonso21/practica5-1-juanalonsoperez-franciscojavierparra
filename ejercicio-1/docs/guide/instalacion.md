# Instalación

Sigue estos pasos para ejecutar el proyecto localmente.

## Requisitos Previos

- Node.js (v20 o superior recomendado)
- npm o yarn

## Pasos

1. Clona el repositorio.
2. Navega a la carpeta `ejercicio-1`.
3. Instala las dependencias:
   ```bash
   npm install
   ```
4. Ejecuta el servidor de desarrollo:
   ```bash
   npm run dev
   ```

## Documentación

Para ver esta documentación localmente:

```bash
npm run docs:dev
```
